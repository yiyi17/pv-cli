使用文档

初始化


pv -v 查看版本号,如果有表示安装成功

npm view pv-cli versions 可以查看所有的版本号

pv init <project-name> <type> 初始化项目

初始化不同项目类型

pv init <project-name> <type>

项目名称 项目的类型

示例: pv init my-project simple或者pv-cli init my-project simple

type值:

simple为vue+webpack3的简单项目,适合简单项目;
webpack4是是simple项目升级到webpack4,适合简单项目;
module是多模块的项目,适合中等大小的项目;
Integrate是module项目的集成,适合多个前端小项目集成到一个项目.

项目的开发

  • type:simple和webpack4

    执行npm run dev启动示例,在浏览器输入http://127.0.0.1:8808就可以访问了

  • type:module和Integrate

    执行npm run dev Test启动示例,Test是模块名称,在浏览器输入http://127.0.0.1:8808就可以访问了,

  • 执行npm run lint修复代码检查
